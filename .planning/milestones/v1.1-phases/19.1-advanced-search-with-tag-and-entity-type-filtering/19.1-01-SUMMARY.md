## Decisions Made
- QueryNode uses 5-variant discriminated union (tag, text, and, or, not) for type-safe AST
- Negated tags handled via `negated` boolean on TagNode; negated freeform text uses NotNode wrapper
- Serializer uses implicit AND (space-separated) rather than explicit "AND" keyword for natural display
- Haversine distance function inlined in evaluator (same implementation as attackStatus.ts) to avoid import coupling
- Range operators support 6 forms: `>N`, `<N`, `>=N`, `<=N`, exact `N`, range `lo-hi`
- Temporal parsing supports both relative (`6h`, `30m`, `2d`, `1w`) and absolute (`2026-03-20`) formats
- Unknown prefixes return false (no match) rather than throwing errors
