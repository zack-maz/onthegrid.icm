  <action>
    Create `server/lib/eventScoring.ts` with:

    1. `GOLDSTEIN_CEILINGS: Record<ConflictEventType, { ceiling: number; downgrade: ConflictEventType | null }>` -- per spec:
       mass_violence/-7/assault, wmd/-7/assault, airstrike/-5/shelling, bombing/-5/shelling, ground_combat/-4/assault, shelling/-4/assault, assassination/-3/assault, abduction/-3/assault, assault/-1/null, blockade/-1/assault, ceasefire_violation/-1/assault.

    2. `applyGoldsteinSanity(entity: ConflictEventEntity): ConflictEventEntity` --
       - If goldsteinScale === 0 OR goldsteinScale > 0: return entity unchanged (skip)
       - Look up GOLDSTEIN_CEILINGS[entity.type]
       - If no entry or downgrade is null: return unchanged
       - If goldsteinScale > ceiling + 3 (i.e., score is LESS negative than expected, exceeding tolerance): return new entity with type set to downgrade target
       - Otherwise return unchanged

    3. `computeEventConfidence(entity: ConflictEventEntity, geoPrecision: 'precise' | 'centroid'): number` --
       Five weighted signals:
       - Media coverage (0.30): `Math.min(1, Math.log2((numMentions ?? 1) + 1) / Math.log2(50))`
       - Source diversity (0.20): `Math.min(1, Math.log2((numSources ?? 1) + 1) / Math.log2(15))`
       - Actor specificity (0.20): both actors non-empty = 1.0, one = 0.5, both empty = 0.0
       - Geo precision (0.15): precise = 1.0, centroid = 0.3
       - Goldstein consistency (0.15): if Goldstein=0 or positive -> 0.5 (neutral). Otherwise: look up GOLDSTEIN_CEILINGS, compute diff = goldsteinScale - ceiling. If diff <= 0 -> 1.0 (within range). If diff > 0 -> Math.max(0, 1.0 - diff/6) (linear decay).
       Return weighted sum.

       - Signal clamping: 1000 mentions still caps media coverage at 1.0
