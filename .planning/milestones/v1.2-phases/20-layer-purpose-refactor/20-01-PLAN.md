<objective>
Remove all 18 entity toggle fields and their functions from uiStore/types, create a new layerStore for visualization layers, and simplify all hooks that consumed toggle state. This is the foundational refactor -- all downstream UI changes depend on this.

Purpose: Entity toggles controlled data visibility, which conflicts with the new "layers = visualization modes" architecture. After this plan, all entities always render, and search/filter is the only way to narrow visible data.
Output: Clean state layer with no entity toggles, new layerStore, simplified hooks.
</objective>
