  <behavior>
    - dispersePosition(centroidLat, centroidLng, globalSlotIndex) returns correct lat/lng for Ring 0 (6 slots at ~3km), Ring 1 (12 slots at ~6km), Ring 2 (18 slots at ~9km)
    - Ring 0 slot 0 is due north of centroid; slots distribute evenly around ring
    - Odd-numbered rings have half-step angle offset to avoid radial alignment
    - Overflow (slot >= 36) wraps to Ring 2 positions
    - Longitude correction applied: dLng accounts for cos(lat) shrinkage at Middle East latitudes (~30-40N)
    - disperseEvents groups events by centroid city (using detectCentroid), sorts by timestamp within each group, assigns slots per-group independently
    - Non-centroid events pass through unchanged
    - DispersionResult includes originalLat, originalLng, dispersedLat, dispersedLng, ringIndex, slotIndex
    - PipelineTrace type includes phaseA checks (rootCode, cameoExclusion, middleEast, geoValid, minSources, actorCountry), phaseB checks (originalType, reclassified, geoPrecision, confidenceSubScores, finalConfidence, passedThreshold), dispersion info, rejection reason
    - AuditRecord wraps event + pipelineTrace + rawGdeltColumns + status (accepted/rejected) + rejectionReason
    - Config additions: eventMinSources (default 2), eventCentroidPenalty (default 0.7), eventExcludedCameo (default ['180','192']), bellingcatCorroborationBoost (default 0.2)
    - Config values load from env vars EVENT_MIN_SOURCES, EVENT_CENTROID_PENALTY, EVENT_EXCLUDED_CAMEO (comma-separated), BELLINGCAT_CORROBORATION_BOOST
  </behavior>

       - `disperseEvents(events: ConflictEventEntity[])` — groups events where actionGeoType is 3 or 4 (new field) by centroid (match using detectCentroid from geoValidation.ts), sorts each group by timestamp, calls dispersePosition per event. Returns events with updated lat/lng and dispersion metadata stored in `data.originalLat`, `data.originalLng` (new optional fields on ConflictEventEntity).

       - Export `buildAuditRecord(...)` helper that assembles an AuditRecord from pipeline state.

       - These store the GDELT ActionGeo_Type value and original centroid coords for dispersed events
