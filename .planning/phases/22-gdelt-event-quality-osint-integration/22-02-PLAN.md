<objective>
Add Bellingcat RSS as the 6th news feed source, implement the event confidence corroboration boost function, and wire the boost into the GDELT event processing pipeline.

Purpose: Bellingcat serves dual roles -- (1) as a high-quality OSINT news source flowing through the existing news pipeline, and (2) as a confidence booster for GDELT events corroborated by Bellingcat investigations. The strict three-gate matching (temporal AND geographic AND keyword) prevents false corroboration. The pipeline integration ensures the boost is actually applied to events end-to-end (EQ-06).
Output: Modified rss.ts with Bellingcat feed, new checkBellingcatCorroboration function in eventScoring.ts, pipeline integration in gdelt.ts, tests
</objective>
