    - m/s to knots: `* 1.94384`
    - m to ft: `* 3.28084`
    - m/s to ft/min: `* 196.85`

    **Data source labels:**
    - Flights: derive from `useFlightStore(s => s.activeSource)` -- map to display names: opensky -> "OpenSky", adsb -> "ADS-B Exchange", adsblol -> "adsb.lol"
    - Ships: hardcoded "AISStream"
    - Events: hardcoded "GDELT v2"

    1. `src/components/detail/FlightDetail.tsx`:
       - Props: `{ entity: FlightEntity }`
       - Sections (each with a small uppercase muted heading):
         - **Identity**: Callsign (or "N/A"), ICAO24, Origin country, Status (Airborne/Ground), Unidentified flag (red text if true)
         - **Position**: Lat/Lng (6 decimal places) -- coordinates are just values here, copy button is in the parent DetailPanelSlot
         - **Movement**: Speed (show both: "{kn} kn / {m/s} m/s"), Heading ("{deg}deg"), Altitude ("{ft} ft / {m} m"), Vertical Rate ("{ft/min} ft/min / {m/s} m/s")
         - **Source**: Data source label, last updated (relative time -- provided by parent)
       - All values rendered via `<DetailValue label="..." value="..." />` or `<DetailValue label="..." value="..." unit="..." />`
       - Handle null values with "--" fallback
       - Read `activeSource` from flightStore for the source label

    2. `src/components/detail/ShipDetail.tsx`:
       - Props: `{ entity: ShipEntity }`
       - Sections:
         - **Identity**: Ship name (or "Unknown"), MMSI
         - **Position**: Lat/Lng (6 decimal places)
         - **Movement**: Speed ("{sog} kn"), Course ("{cog}deg"), Heading ("{hdg}deg")
         - **Source**: "AISStream", last updated (from parent)

    3. `src/components/detail/EventDetail.tsx`:
       - Props: `{ entity: ConflictEventEntity }`
       - Sections:
         - **Event**: Type label (Drone/Missile), Event type, Sub-event type, CAMEO code, Goldstein scale
         - **Location**: Location name, Lat/Lng (6 decimal places)
         - **Actors**: Actor 1, Actor 2 (show "--" if empty)
         - **Source**: "GDELT v2", Date (from entity.timestamp formatted as YYYY-MM-DD). Source article link: render as a small inline `<a>` tag with blue accent color, target="_blank", text "View source". Only show if `data.source` is non-empty.

    All three components should have consistent section styling: section heading is `text-[10px] uppercase tracking-wider text-text-muted mb-1 mt-3 first:mt-0`, values are stacked vertically with gap-1.
