## Accomplishments
- Created RadialGradientExtension with GLSL fragment shader injection for radial alpha falloff (hot center fading to transparent edge)
- Replaced 8-stop FLIR Ironbow palette with 4-stop simplified thermal (deep purple to bright red)
- Decoupled radius (encodes geographic spread via cell count, 12-100px) from color (encodes threat weight via P90 normalization)
- Added additive blending parameters for natural overlap intensification
- Implemented hover dimming: hovered cluster brightens to 255 alpha, non-hovered dim to 40% (102 alpha)
- Added zoom-dependent layer z-order: threats on top at zoom < 9, behind entity markers at zoom >= 9
- Zoom threshold tracked with ref-based crossover detection (zero 60fps re-renders)

## Decisions Made
- Used `smoothstep` GLSL function for radial falloff (smooth hermite curve -- natural thermal appearance, good middle ground between linear and quadratic)
- Linear interpolation for cell-count-to-pixel radius (plan specified linear, alternative sqrt considered but not used)
- Retained P90 normalization from Phase 23 for color domain (prevents high-weight outliers from washing out)
- Tracked zoom threshold as boolean (`isBelowZoom9`) via ref-based crossover detection instead of continuous zoom value (prevents 60fps re-renders during smooth zoom per research pitfall 5)
- Managed hoveredClusterId as local React state in BaseMap rather than uiStore (transient visual state, not persistent)
- Used `onMove` callback with `e.target.getZoom()` for threshold detection (confirmed available on react-maplibre Map)
