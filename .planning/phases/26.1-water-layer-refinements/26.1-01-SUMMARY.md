## Decisions Made
- isPriorityCountry uses duplicated 29-country centroids from basinLookup.ts (avoids circular dependency between adapter and lib modules)
- Fixed isExcludedLocation to use the full 29-country COUNTRY_CENTROIDS_FULL array instead of the sparse 5-entry CENTROIDS -- the sparse array was incorrectly resolving Iran/Pakistan/etc. to Uzbekistan due to nearest-neighbor on too few data points
- treatment_plant uses diamond icon placeholder in frontend (same as dam) pending dedicated water icons
- Cron refresh=true parameter guarded by vercel-cron user-agent in production; dev always allows refresh
