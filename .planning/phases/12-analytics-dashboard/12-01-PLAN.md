    **1. Create `src/components/counters/useCounterData.ts`:**

    Export a `CounterValues` interface:
    ```typescript
    export interface CounterValues {
      iranianFlights: number;
      unidentifiedFlights: number;
      airstrikes: { filtered: number; total: number };
      groundCombat: { filtered: number; total: number };
      targeted: { filtered: number; total: number };
      totalEvents: { filtered: number; total: number };
      fatalities: { filtered: number; total: number };
    }
    ```

    Export `useCounterData()` hook that:
    - Reads `rawFlights` from `useFlightStore(s => s.flights)` and `rawEvents` from `useEventStore(s => s.events)`
    - Reads toggle state from `useUIStore`: `showEvents`, `showAirstrikes`, `showGroundCombat`, `showTargeted` (individual scalar selectors)
    - Calls `useFilteredEntities()` to get smart-filter-passing entities
    - Computes flight counters from `rawFlights` directly (no filter/toggle narrowing per user decision):
      - `iranianFlights`: `rawFlights.filter(f => f.data.originCountry === 'Iran').length`
      - `unidentifiedFlights`: `rawFlights.filter(f => f.data.unidentified).length`
    - Computes event totals from `rawEvents` (unfiltered):
      - For each group, count events matching `CONFLICT_TOGGLE_GROUPS[key]` using `.includes()` with `as readonly string[]` cast
    - Computes event filtered counts by applying BOTH smart filters (from `useFilteredEntities().events`) AND toggle gating:
      - If `showEvents && showAirstrikes`, count smart-filtered events matching showAirstrikes group; else 0
      - Same for groundCombat (showEvents && showGroundCombat) and targeted (showEvents && showTargeted)
    - Computes totalEvents as sum of three groups (independently for filtered and total)
    - Computes fatalities.total as `rawEvents.reduce((sum, e) => sum + e.data.fatalities, 0)`
    - Computes fatalities.filtered as sum of fatalities from smart-filtered + toggle-gated events
    - Wrap all computations in `useMemo` with appropriate deps
    - Use `Intl.NumberFormat('en-US')` for number formatting (returned as raw numbers; formatting happens in CounterRow)

    Component:

    - Delta display: `<span className="text-accent-green text-xs animate-delta">+{delta}</span>` (or `-{delta}` for negative). Use a key prop tied to a render counter to restart the CSS animation on each new delta.
    - Format numbers with `Intl.NumberFormat('en-US').format()`
