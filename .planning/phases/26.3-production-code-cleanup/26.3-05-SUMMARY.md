## Decisions Made

- **noUncheckedIndexedAccess on server only** — Client-side strict index access would cascade through deck.gl/maplibre layer code where runtime types are looser than declared. Server has no such constraints and benefits most from the flag.
- **`getCol()` helper for GDELT CSV** — Centralizes bounds-checked column reads rather than scattering non-null assertions across the adapter; makes the column access pattern auditable.
- **Cast-with-comment for deck.gl v9** — `GeoJsonLayer` and `IconLayer` accept `FeatureCollection`/`HTMLCanvasElement` at runtime but v9 type defs are stricter than the runtime accepts. `as any` with inline comment is the least invasive fix that preserves full type safety elsewhere.
- **Rate limit test fixture swap** — Instead of preserving the deprecated `rateLimitMiddleware` export purely for test convenience, the test suite now uses `rateLimiters.flights` as its canonical fixture through a local const alias. The describe block name (`'rateLimitMiddleware'`) is retained purely for human readability.
- **OpenAPI hand-written vs. generated** — Hand-writing avoids adding a code-gen runtime dependency (e.g. `@asteasolutions/zod-to-openapi`), and editorial descriptions on each endpoint carry more signal for portfolio review than mechanically derived text.
- **`allOf` composition for CacheResponse** — OpenAPI 3.0 doesn't have generics, so the shared `CacheResponseBase` (with `stale`, `lastFresh`, `degraded?`) is composed via `allOf` with per-endpoint `data` schemas rather than duplicating the envelope.
