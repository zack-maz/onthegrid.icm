## Decisions Made

- **res.locals.validatedQuery pattern:** Express 5 makes req.query a read-only getter property. Attempting `req.query = result.data` throws "Cannot set property query of IncomingMessage which has only a getter". Validated data is stored on res.locals.validatedQuery instead.
- **Zod v3 (not v4):** npm installed zod@latest which got v4.3.6, but v4 removes ZodTypeAny and has breaking API changes. Pinned to zod@3 for stable API compatibility.
- **importOriginal mock pattern:** All 14 test files that mock config.js updated to use `vi.mock('../../config.js', async (importOriginal) => { const actual = await importOriginal(); return { ...actual, config: mockCfg, ... }; })`. This preserves the constant re-exports (WAR_START, CACHE_TTL, etc.) that are now part of config.js.
- **Test env defaults:** parseEnv() uses explicit if-checks (`if (!merged.KEY) merged.KEY = 'default'`) instead of spread pattern to avoid env var override order bugs where process.env spread overwrites test defaults.
