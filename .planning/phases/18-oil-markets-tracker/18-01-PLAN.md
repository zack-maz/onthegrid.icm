<task type="auto">
  <name>Task 2: Yahoo Finance adapter, cache-first route, and server wiring</name>
  <files>server/adapters/yahoo-finance.ts, server/routes/markets.ts, server/index.ts</files>
  <action>
    **Create `server/adapters/yahoo-finance.ts`:**

    - Define internal `YahooChartResponse` interface matching the v8 chart API response shape (chart.result[].meta + chart.result[].indicators.quote + chart.result[].timestamp)
    - Export `TICKERS = ['BZ=F', 'CL=F', 'XLE', 'USO', 'XOM'] as const`
    - Export `DISPLAY_NAMES: Record<string, string>` mapping BZ=F -> "Brent", CL=F -> "WTI", rest same as ticker
    - Implement `fetchTicker(symbol: string): Promise<MarketQuote | null>`:
      - URL: `https://query2.finance.yahoo.com/v8/finance/chart/${encodeURIComponent(symbol)}?range=5d&interval=1d&includePrePost=false`
      - Headers: `{ 'User-Agent': 'Mozilla/5.0 (compatible)' }` to avoid bot detection
      - Parse response, extract meta.regularMarketPrice, meta.previousClose (fallback to meta.chartPreviousClose), meta.currentTradingPeriod.regular for marketOpen
      - Filter null values from close/high/low arrays
      - Convert timestamps from Unix seconds to ms
      - Return null on any failure (non-ok status, missing data, parse error)
    - Export `fetchMarkets(): Promise<MarketQuote[]>`:
      - Use `Promise.allSettled(TICKERS.map(fetchTicker))`
      - Filter for fulfilled results with non-null values
      - Return array of MarketQuote (may be 0-5 items depending on per-ticker success)

    **Create `server/routes/markets.ts`:**

    - Follow the exact cache-first pattern from ships.ts and news.ts
    - `const MARKETS_KEY = 'markets:yahoo'`
    - Import `MARKETS_CACHE_TTL`, `MARKETS_REDIS_TTL_SEC` from constants
    - GET `/` handler:
      1. `cacheGet<MarketQuote[]>(MARKETS_KEY, MARKETS_CACHE_TTL)` -- return if fresh
      2. On miss/stale: `fetchMarkets()` -> `cacheSet(MARKETS_KEY, quotes, MARKETS_REDIS_TTL_SEC)` -> return `{ data, stale: false, lastFresh: Date.now() }`
      3. On upstream error: return stale cache with `stale: true` if available, else 500

    **Wire in `server/index.ts`:**

    - Import `marketsRouter` from `./routes/markets.js`
    - Add `app.use('/api/markets', marketsRouter)` after the existing news route
