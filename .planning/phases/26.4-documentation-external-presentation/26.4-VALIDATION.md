## Sampling Rate

- **After every task commit:** Run the relevant subset (`npx vitest run <path>` for the files touched)
- **After every plan wave:** Run `npx vitest run` + `npx tsc -b` + `npx type-coverage` (once installed)
- **Before `/gsd:verify-work`:** Full suite green, CI workflow passing on a PR, gitleaks passing on a planted-secret fixture
- **Max feedback latency:** 30 seconds for task-level, 90 seconds for wave-level
