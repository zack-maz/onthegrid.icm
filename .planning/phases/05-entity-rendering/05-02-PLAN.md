1. Update `src/components/map/layers/constants.ts`:
   - Replace the flat `ICON_SIZE` object with structured sizing that includes meter-based size and pixel bounds:
     ```typescript
     export const ICON_SIZE = {
       flight: { meters: 800, minPixels: 6, maxPixels: 40 },
       ship:   { meters: 600, minPixels: 5, maxPixels: 36 },
       drone:  { meters: 800, minPixels: 6, maxPixels: 40 },
       missile:{ meters: 800, minPixels: 6, maxPixels: 40 },
     } as const;
     ```
   - The `meters` value controls the real-world footprint: ~800m means a chevron spans ~800m of ground. At zoom 5 (Iran overview, ~150km wide), 800m is ~3px. At zoom 10 (~5km wide), 800m is ~100px -- clamped to maxPixels 40. This gives a clear scaling ramp between zoom 5-10.
   - `minPixels: 6` prevents icons from vanishing at continental zoom (3-4).
   - `maxPixels: 40` prevents icons from becoming huge blobs at street zoom (12+).

2. Update `src/hooks/useEntityLayers.ts`:
   - For ALL 4 IconLayer instances (flights, ships, drones, missiles), change:

     - Add `sizeMaxPixels: ICON_SIZE.{type}.maxPixels`
