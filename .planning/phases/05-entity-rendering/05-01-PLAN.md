    **Create `src/hooks/useEntityLayers.ts`:**

    Import `IconLayer` from `@deck.gl/layers`, `useFlightStore` from `@/stores/flightStore`, `useUIStore` from `@/stores/uiStore`, constants from `@/components/map/layers/constants`, and `getIconAtlas`/`ICON_MAPPING` from `@/components/map/layers/icons`. Import types `FlightEntity`, `ShipEntity`, `ConflictEventEntity` from `@/types/entities`.

    Export `useEntityLayers()` hook that:

    1. Reads `flights` from useFlightStore via selector `s => s.flights`.
    2. Reads `pulseEnabled` from useUIStore via selector `s => s.pulseEnabled`.
    3. Manages pulse animation state:
       - Use `useRef<number>` for the animation frame ID.
       - Use `useState<number>(1.0)` for pulseOpacity, BUT throttle updates to ~15fps (every ~67ms) to avoid excessive re-renders. Inside the rAF loop, only call `setPulseOpacity` when the elapsed time since last update exceeds 67ms.
       - Pulse formula: `opacity = PULSE_CONFIG.minOpacity + (PULSE_CONFIG.maxOpacity - PULSE_CONFIG.minOpacity) * (0.5 + 0.5 * Math.sin(Date.now() / PULSE_CONFIG.periodMs * Math.PI * 2))` -- oscillates 0.7-1.0 over 2 seconds.
       - Start rAF loop in useEffect, cleanup with cancelAnimationFrame.
       - Only run the animation loop when `pulseEnabled` is true. When false, set pulseOpacity to 1.0.
    4. Build layers using `useMemo`:

       **Flight IconLayer:**
       - `id: 'flights'`
       - `data: flights`
       - `iconAtlas: getIconAtlas()`
       - `iconMapping: ICON_MAPPING`
       - `getIcon: () => 'chevron'`
       - `getPosition: (d) => [d.lng, d.lat]`
       - `getSize: ICON_SIZE.flight`
       - `sizeUnits: 'pixels' as const`

       - `getColor: (d) => { const [r,g,b] = d.data.unidentified ? ENTITY_COLORS.flightUnidentified : ENTITY_COLORS.flight; const alpha = d.data.unidentified ? Math.round(pulseOpacity * 255) : Math.round(altitudeToOpacity(d.data.altitude) * 255); return [r, g, b, alpha]; }`
       - `billboard: false` -- rotate with map tilt
       - `updateTriggers: { getColor: [pulseOpacity] }`
